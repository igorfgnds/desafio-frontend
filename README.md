## Desafio Frontend
Esta aplicação foi desenvolvida como solução para o Desafio Frontend das empresas GOinside e Supero Tecnologia.

A principal função da aplicação consiste em consultar o webservice do ADSim (https://api.adsim.co/crm/api/v1/refrigerante/listar) para listar os refrigerantes retornados pelo endpoint.

É possível "selecionar" os refrigerantes e "curtir" os seus favoritos. Foi criado um "carrinho" para listar instantaneamente os refrigerantes selecionados, bem como a quantidade total e o preço total de um possível "pedido" de compra. Também existe a possibilidade de limpar a lista selecionada através do botão "Limpar carrinho".

### Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Sass
- React ([Create React App](https://facebook.github.io/create-react-app/ "Create React App"))
- Bootstrap 4 ([React Bootstrap](https://react-bootstrap.github.io/ "React Bootstrap"))

### Pré-requisitos
- [Node.js](https://nodejs.org "Node.js") >= 8.10 instalado

### Como executar
Para executar a aplicação em ambiente local, clone este repositório, abra o diretório do projeto no terminal e execute os seguintes comandos:

##### Instalar dependências do projeto:
```sh
yarn install
ou
npm install
```

##### Executar projeto em modo de desenvolvimento:
```sh
yarn start
ou
npm start
```
